# Smart Room Climate Monitor
A live temperature & humidity display that warns you when your room gets too hot or humid.
### Components Required
ESP32 Dev Board + USB cable
DHT11 Temperature & Humidity Sensor
OLED Display 0.96" I2C (SSD1306)
Active Buzzer
LEDs (Red + Green) + 220Ω resistors
Breadboard + Jumper Wires
### Library Versions
Adafruit_SSD1306
dafruit_GFX
DHT
### Expected Output
The DHT11 sensor will detect the current temperature and humidity and send a signal to the ESP32. A live temperature & humidity will be displayed on the OLED.
