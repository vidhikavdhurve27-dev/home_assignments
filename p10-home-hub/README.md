# Project 10: Full IoT Home Automation Hub
Combine everything you've learned — sensors, display, relay, Wi-Fi, and MQTT — into one complete smart home system
### Components Required
ESP32 Dev Board \
DHT11 Temperature & Humidity Sensor \
BMP180 Gas Sensor \
PIR Motion Sensor HC-SR501 \
LDR + 10kΩ resistor \
2-Channel Relay Module// Single Channel Relay (simulate fan + light) \
OLED Display 0.96" I2C \
Active Buzzer + LEDs (Red, Green, Yellow) \
Push Buttons ×2 (manual override) \
Breadboard + Jumper Wires \
