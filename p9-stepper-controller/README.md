# Project 9: Stepper Motor Precision Positioner
Control a stepper motor with precise step control — the foundation of CNC machines, 3D printers, and robotic arms.
###  Components Required
Arduino Uno R3 \
Stepper Motor 28BYJ-48 + ULN2003 Driver Board \
Potentiometer 10kΩ (position control) \
Push Buttons ×3 (CW, CCW, Set Home) \
OLED Display 0.96" I2C \
LED (position indicator) + resistors \
Breadboard + Jumper Wires \
